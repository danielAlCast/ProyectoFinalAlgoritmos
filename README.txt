MergeSort:

##Para compilar el programa MergeSort
#Escribir:  javac *.java

##Para ejecutar el programa MergeSort y pasarle un archivo por redireccionamiento
#Escribir:  java mS < pruebas/test3.txt

##Para obtener el resultado de la ejecución en un archivo de texto
#Escribir:  java mS < pruebas/test3.txt > resultado5.txt
